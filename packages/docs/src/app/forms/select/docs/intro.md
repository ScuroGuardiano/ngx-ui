You'll need to use the `ngxInputAttribute` directive to add the following inputs to the select (or any input) components

- label
- required
- hint
- placeholder
- minWidth
- tabIndex
- min
- max
- maxLength
- minLength
- disabled
- autocorrect
- spellcheck
- passwordToggleEnabled
- passwordTextVisible
- unlockable
- unlockableToolTip
- type
